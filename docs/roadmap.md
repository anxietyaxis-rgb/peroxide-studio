# Roadmap (Draft)

## Phase 0 — Foundation (now)
- Public repo + clean documentation
- Clear rules and anti-abuse policy
- Define verification sources per game/platform
- Establish community roles and responsibilities

## Phase 1 — Prototype
- Basic “Proof-of-Play” submission format
- Moderator verification workflow
- Public leaderboard (reputation, not money)
- Simple reward rules (manual payouts)

## Phase 2 — Automation
- Lightweight bot to ingest proof + assign roles
- Fraud detection + rate limiting

## Phase 3 — Public Good Expansion
- Sponsor and creator partnerships
- Transparent community pools
- Events/tournaments tied to verifiable play
- Optional on-chain attestations (only if it makes sense)
